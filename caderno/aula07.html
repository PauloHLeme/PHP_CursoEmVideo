Operadores Relacionais

Como fazer comparações em PHP

Os operadores relacionais, como o nome diz, serve para compararmos valores entre variáveis.

  • Maior que: >
  • Menor que: <
  • Maior ou Igual a: >=
  • Menor ou Igual a: <=
  • Diferente de: <> ou !=
  • Igual a: ==
  • Idêntico a: ===

Repare que temos o sinal que representa igualdade e o que representa igualdade exclusiva, também conhecido como idêntico a. A diferença é que o primeiro vai comparar se os valores são iguais, independente do tipo de variável. O segundo só vai retornar verdadeiro para a igualdade se os valores forem iguais E a variável for do mesmo tipo.

Se, por exemplo tivermos a variável $a = 2 e a variável $b = '2', a é uma variável tipo number e b tipo string. O sinal de "Igual a" vai retornar verdadeiro, mas o "Idêntico a" retornará falso.

Operador Ternário

O operador ternário verifica, em uma linha de código, se uma expressão é verdadeira ou falsa, e retorna um valor para cada alternativa. Para tanto, escrevemos uma expressão lógica para ser testada, seguida do sinal de ?, logo depois colocamos o retorno caso a expressão seja verdadeira, adicionamos o sinal : e por fim, o retorno caso falso.

Exemplo


    $maior = $a > $b ? $a : $b;

Aqui temos que, se a for maior que b, retornaremos o valor de a para dentro da variável maior, do contrário, retornaremos o valor de b.

Operadores lógicos

São operadores que verificam expressões do estilo E e OU. Operações comuns executadas por computadores incontáveis vezes por dia!

  • ! Negação (NÃO lógico)
  • and ou && Conjunção (E lógico)
  • or ou || Disjunção (OU lógico)
  • xor Disjunção Exclusiva (XOU lógico)

Operações Lógicas


    Negação             Resultado
    !true               false
    !false              True


    Conjunção           Resultado
    true && true        True
    true && false       false
    false && true       false
    false && false      false


    Disjunção           Resultado
    true || true        True
    true || false       True
    false || true       True
    false || false      false

    Disjunção           Resultado
    true || true        false
    true || false       True
    false || true       True
    false || false      false
