Estrutura Condicional Switch

Condições de múltiplos casos

Se precisarmos testar várias situações para uma mesma variável, ou seja, se tivermos várias possibilidades de retorno, temos a estrutura switch.

Exemplo


    switch ($op){
        case 1:
            echo "A";
            break;
        case 2:
            echo "B";
            break;
        default:
            Echo "Erro";
    }

Ao ilustrarmos o exemplo, colocamos apenas dois comandos case, mas podemos utilizar quantos forem necessários. Ao final ainda, podemos fazer uso do comando default, que serve para retornar uma mensagem caso nenhum dos testes seja a solução.

Caso o valor da variável op fosse 1, o código executaria apenas o case de cima, pois é o correspondente. Caso a variável valesse 2, executaria apenas o segundo, pulando o primeiro, e caso fosse outro valor, executaria o comando default.

Dica

Perceba que após cada case, temos um comando break. Ele serve para parar a execução do código switch, pois já achamos a linha desejada.

Caso não colocássemos o break, o código começaria da linha pertinente a execução, mas continuaria rodando para todas as linhas abaixo dele.

Dessa forma, é importante colocar o break ao final de cada case.

Condições de Múltiplos casos

Temos ainda a opção de executar uma saída única para mais de um case, caso seja necessário. Nesse caso, a sintaxe se dá da seguinte maneira:


    Switch ($op){
        case 1:
        case 2:
        case 3:
            echo "A";
            break;
        case 4:
        case 5:
            echo "B";
            break;
        default:
            echo "C";
    }
