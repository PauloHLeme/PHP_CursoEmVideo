Operações Aritméticas

Os operadores aritméticos são utilizados para fazerem cálculos simples, assim como a matemática no dia a dia.

Operadores Aritméticos

Em PHP, os operadores são bem simples:

  • Soma: +
  • Subtração: -
  • Multiplicação: *
  • Divisão: /
  • Módulo (resto de divisão): %

Dica!

Vale lembrar que se colocarmos o operador dentro de uma concatenação, ele não assumirá a função de operador, mas será exibido como um caractere comum. Dessa forma, para representarmos a soma de $n1 + $n2, sendo n1 = 2 e n2 = 3, temos que escrever: "A soma vale". ($n1+$n2), assim então recebendo como retorno A soma vale 5. Do contrário não conseguiríamos somar dentro da concatenação. Então, se escrevêssemos "A soma vale $n1 + $n2", teríamos como saída "A soma vale 2+3".

Ordem de Precedência

Se tivermos vários operadores em uma mesma expressão, eles possuem uma ordem de precedência que deve ser seguida, e ela é exatamente igual ao da matemática do dia a dia. Ou seja

  1. Parênteses: ()
  2. Multiplicação, divisão e módulo: * / %
  3. Soma e subtração: + -

Valores Personalizados

Em PHP, temos a capacidade de atribuirmos valores personalizados para a variável direto pela URL do nosso navegador. Traduzindo, podemos colocar lá no caminho que aparece no topo do nosso navegador, logo após o nome do arquivo, alguns dados que vão ditar para o código qual valor cada variável vai receber.

Para isso, temos dois passos básicos para seguir:

Primeiro, devemos fazer as nossas variáveis receberem valores externos. Isso é feito pelo comando $_GET[""].

Após isso, vamos no final da nossa URL, e digitamos os valores que queremos da seguinte forma ?a=3. Vamos ao exemplo:

Exemplo

Vamos considerar que nossa URL é:

http://localhost/aula05/operadores.php

Nesse caso, para definirmos os valores de duas variáveis como 3 e 2, colocaríamos no final:

http://localhost/aula05/operadores.php?a=3&b=2

Enquanto isso, dentro do nosso código, as variáveis não recebem um valor fixo, ao invés disso, receberão o comando _GET[""] para ficarem preparadas para os valores que virão:


    $n1 = _GET["a"];
    $n2 = _GET["b"];

Pronto, agora ao executarmos o arquivo adicionando as variáveis desejadas no final da URL, teremos os cálculos feitos com as variáveis de acordo com os valores que passamos!

Essa forma não é ainda interativa com o usuário, mas serve para trabalharmos com testes com alterações de variáveis.

Outras operações matemáticas

Para os demais cálculos matemáticos não cobertos pelas operações aritméticas, o PHP conta com várias funções específicas que nos fornecem os cálculos necessários. São alguns deles, por exemplo:

  • abs(): Retorna o valor absoluto, o módulo do valor, sem sinal de negativo, por exemplo
  • pow(): Potenciação
  • sqrt(): Raiz quadrada
  • round(): Arredondamento matemático padrão
  • ceil(): Arredonda sempre para cima (teto)
  • floor(): Arredonda sempre para baixo (chão)
  • intval(): Pega apenas a parte inteira de um valor quebrado
  • number_format(): utilizado para formatar números, colocando a quantidade máxima de casas decimais, por exemplo. Ou ainda trocar o . por ,

Exemplo


    $v1 = $_GET["x"];
    $v2 = $_GET["y"];

    echo "Valores recebidos: $v1 e $v2";
    echo "O valor absoluto de $v2 é igual a ". abs($v2);
    echo "<br>O valor de $v1 <sup>$v2</sup> é ". pow($v1, $v2);
    echo "<br>A raiz de $v1 é ". sqrt($v1);
    echo "<br>O valor arredondado de $v2 é ". round($v2);
    echo "<br>A parte inteira de $v2 é ". intval($v2);
    echo "<br>O valor de $v2 em moeda é 
          R$ ". number_format($v2,2,',','.');

Dessa forma, se definirmos na URL que ?x=4&y=3.5, a saída será:

    
    Valores recebidos: 4 e 3.5
    O valor absoluto de 3.5 é igual a 3.5
    O valor de 4 3.5 é 128
    A raiz de 4 é 2
    O valor arredondado de 3.5 é 4
    A parte inteira de 3.5 é 3
    O valor de 3.5 em moeda é R$ 3,50
