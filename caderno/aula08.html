Integração em HTML5 + PHP

Formulários em HTML5

Todo formulário em HTML é executado dentro da tag form. Dentro da tag temos que ter os parâmetros method = "" e action = "". O primeiro indica a forma de captura dos dados, que pode ser get ou post e o segundo marca o arquivo onde armazenaremos os dados, que é um arquivo em PHP.

Dentro do formulário, temos inputs que servirão para inserirmos os valores e um input do tipo submit, que serve para enviarmos os dados justamente para o arquivo que especificamos dentro do parâmetro action.

É importante que todos os inputs de entrada de valor do formulário tenham também o atribulo name= "", pois o PHP vai fazer uso deles.

Exemplo

Abaixo, temos o exemplo do código HTML:


    <form method="get" action="dados.php">
        <label for="v">Valor</label><input type="text" name="v">
        <input type="submit" value="Calcular" >
    </form>

Aqui, utilizamos o método get para a coleta dos dados. Ele é similar ao que vínhamos fazendo até agora, ao digitar os valores da variável no fim da URL do site, mas neste caso, a página fará isso pra gente ao apertarmos o botão Calcular.

Agora, temos o que irá internamente no código PHP:


    <?php
        $valor = $_GET["v"];
        echo "Digitou $valor.";
    ?>

Aqui, para pegarmos o valor da variável, como utilizamos o method= "get", temos que utilizar do $_GET para puxar o valor. Se ao invés disso, tivéssemos utilizado o method= "post", faríamos uso do $_POST.

Utilizando outros tipos de controle

Podemos utilizar como entrada todos os tipos de dados de controle que existem nos formulários. Dessa forma, podemos por exemplo checkbox, listas de seleção, etc. Temos entretanto que ficar de olho pois, algumas delas, como as radio por exemplo, temos que inserir um parâmetro value="" onde vamos dizer qual o valor que queremos que seja retornado.

CSS editado com PHP

O código php não precisa ser feito somente dentro do body de um HTML, podemos também fazer ele no Head, e assim criar as variáveis antes de criar o style do seu site.

Dessa forma, podemos criar variáveis que recebam valores específicos para as CSS.

Exemplo


    <head>
        <?php
            $txt = isset($_GET['t'])? $_GET['t']: 'lorem ipsum';
            $tam = isset($_GET['tam'])? $_GET['tam']: '12pt';
            $cor = isset($_GET['cor'])? $_GET['cor']: '#000000';
        ?>
        <style>
            span.texto{
                font-size: <?php echo $tam; ?>;
                color: <?php echo $cor; ?>;
            }
        </style>
    </head>

    <body>
        <?php
        echo "<span class= 'texto'>$txt</span>";
        ?>
    </body>

Nesse exemplo, temos a tag ?php ? sendo utilizada quatro vezes:

  • Na primeira, dentro do head e antes do style, para criarmos as variáveis que serão utilizadas.
  • Na segunda e terceira vez, para colocarmos o valor definido pelo PHP nos atributos font-size e color.
  • E no quarto e último uso, já dentro do body, para chamar a variável txt.
