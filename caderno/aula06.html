Operadores de Atribuição

Atribuição é quando você insere, atribui um valor para dentro de uma variável. Os operadores de atribuição são os sinais utilizados para inserir o valor para a variável

Quais são os operadores de operação?

  • Adição: + ou +=
  • Subtração: - ou -=
  • Multiplicação: * ou *=
  • Divisão: / ou /=
  • Módulo: % ou %=
  • Concatenação: .

Exemplo


    $a = $a + b     ou      $a += $b
    $a = $a - b     ou      $a -= $b
    $a = $a * b     ou      $a *= $b
    $a = $a / b     ou      $a /= $b
    $a = $a % b     ou      $a %= $b
    $a = $a . b     ou      $a .= $b

No exemplo da soma, a variável a está recebendo (=) o valor de a + b. Podemos tanto escrever a forma completa, que é a primeira coluna, quanto a forma resumida, da segunda coluna.

Operadores de Incremento

Os operadores de incremento servem exatamente para o que o nome deles diz, incrementar o valor da variável. Existem dois tipos e cada um pode ser utilizado em duas situações:

  • Pré Incremento: $a = $a +1 ou $a++
  • Pós Incremento: $a = $a +1 ou ++$a
  • Pré Decremento: $a = $a -1 ou $a--
  • Pós Decremento: $a = $a -1 ou --$a

A utilização do pré ou pós incremento varia de quando desejamos que a alteração ocorra. Se quisermos utilizar a variável no valor original, e depois realizarmos o incremento, utilizamos o pós incremento. Caso seja necessário incrementar o valor antes de o utilizar, fazemos uso do pré.

Comentários

Comentários são essenciais para nos lembrarmos no futuro sobre o que fizemos nos nossos códigosp presentes ou até mesmo para explicar para outros desenvolvedores que pegarem seu código.

Em PHP podemos utilizar:

  • // : Tudo o que for escrito depois das duas barras, mas na mesma linha, serão dados como comentários
  • # : Funciona igual ao comentário anterior
  • /* */ : Tudo o que estiver dentro desses símbolos, serão interpretados como comentários, independente de quantas linhas estejam dentro deles.

Referências entre Variáveis

Podemos utilizar uma variável para servir como entrada de valor para outra variável, como já vimos anteriormente. Mas ao mudar o valor de uma delas, a outra permanece inalterada.

O PHP nos permite ligar duas variáveis, de forma que uma referencie a outra. Dessa forma, se fizermos alteração em uma, a outra também vai receber tal mudança.

Para isso, utilizamos o &

Exemplo


    $a = 3;
    $b = 5;

    //sem referenciais         //com referenciais
    $b = $a;                   $b = &$a;
    $b += 5;                   $b += 5;

Nos exemplos acima, se não utilizarmos a chamada referencial, e pedirmos para retornar os valores de a e b, teremos que a vale 3 e b vale 8.

Já com o uso de referencial, tanto a quanto b agora valem 8! Com um único cálculo, conseguimos transformar as duas variáveis, pois elas estão ligadas.

Variáveis de variáveis

Esse é um conceito pouco utilizado no dia a dia, mas existente no PHP.

Exemplo


    $site = "cursoemvideo";
    $$site = "cursoPHP";

    echo $site;
    echo $cursoemvideo;

Olhe bem para o exemplo acima. A primeira variavel, de nome site foi criada normalmente. Na criação da segunda, foi utilizado dois sinais de $$. Dessa forma, foi criada uma variável de uma variável. Não criamos uma nova variável site, mas sim uma variável que vai receber, como nome, o valor da "variável pai".

Por isso que no nosso exemplo, ao darmos o comando echo, temos uma variável site e uma variável nova chamada cursoemvideo. Pois ela foi gerada com o nome contido no valor da variável anterior.
