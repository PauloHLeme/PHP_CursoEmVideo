Como criar condições em PHP

Condições são situações onde o código se divide em duas condições distintas, que serão executadas ou não de acordo com uma condição lógica que definirmos.

É como se tivéssemos dois blocos de códigos diferentes, para duas situações diferentes. Se a condição der o resultado a, executamos o primeiro bloco e não executamos o segundo, do contrário, executamos o segundo e o primeiro é quem fica inativo.

Para representarmos as condições em PHP, temos a função if.

Exemplo

Condições Aninhadas

Existirão casos, váaaaaarios casos, onde uma única condição não basta, pois temos mais de dois caminhos.

Nesse caso, temos a possibilidade de realizar aninhamentos de condições, isso é, colocar condições if dentro de outras condições if anteriores e irmos bifurcando os caminhos conforme necessário.

Exemplo

No nosso exemplo anterior, com a votação, não colocamos por exemplo a condição do voto opcional, que acontece quando a pessoa tem entre 16 e 18 anos.

Para solucionarmos essa questão, podemos fazer uso das condições aninhadas!


    if ($idade <16){
        $voto = "não vota";
    }else{
        if($idade >=16 && <18){
            $voto = "voto opcional";
        }else{
            $voto = "voto obrigatório";
        }
    }

Pronto! Agora temos que, SE a pessoa for menor de 16 anos, ela não pode votar. SENÃO, existem duas possibilidades: SE tiver entre 16 e 18 anos, o voto é opcional, ou SENÃO, obrigatório. Por isso colocamos o segundo if dentro do primeiro else, pois aninhamos essa bifurcação apenas se a primeira for falsa.

Dica

Temos ainda o comando elseif, que adiciona um comando if logo em seguida do else, eliminando um bloco extra.

Exemplo


    if ($idade <16){
        $voto = "não vota";
    }elseif($idade >=16 && <18){
        $voto = "voto opcional";
    }else{
        $voto = "voto obrigatório";
    }

Percebeu só? Ao juntarmos os comandos em um elseif, conseguimos diminuir o nosso código e ter o mesmo resultado!
