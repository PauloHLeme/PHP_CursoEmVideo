Variáveis em PHP

De início, com os exemplos dados em aula, aprendemos o comando echo, que é utilizado para exibir o que se deseja na tela.

Dentro do comando echo, podemos colocar tanto sentenças quanto tags em HTML

Tá, mas e o que são as variáveis?

As variáveis podem ser explicadas como sendo uma espécie de armário, que contém espaços, e o espaço pode receber alguma coisa.

Dessa forma, se eu crio um espaço chamado idade, posso colocar nele um valor de 3.

Em PHP, declaramos a variável com o símbolo $, em seguida damos um nome em letras minusculas e sem acentuação, o sinal de igual para atribuirmos um valor, que será escrito logo em seguida, e para fechar a linha, devemos terminar com o ;.

Exemplo


    $idade = 3;
        ou,
    $sal = 1825.54;
        ou ainda,
    $nome = "Leonardo";
        ou também,
    $casado = false;

Nesses exemplos, cada variável tem um valor diferente. A idade é um tipo de dado Inteiro, o salário é Real, o nome é Caractere e o casado é Lógico.

O PHP trabalha os tipos de dados dinamicamente, sem ser necessário declarar o tipo de cada variável, mas podemos forçar uma conversão com uma declaração conhecida como typecast.

  • Inteiro: (int),(integer)
  • Real: (real), (float), (double)
  • Caractere: (string)
  • Lógico: Não existe, pois variáveis lógicas recebem apenas o valor 1 se forem true ou ficam vazias se forem false

Como fazer Concatenação?

Concatenação é realizar a exibição de mensagens misturando texto e variáveis.

Para isso, o PHP também nos facilita bastante, pois se colocarmos a saída entre aspas duplas, a concatenação vai ser feita de forma muito simples.

Exemplo


    $idade = 28;
    $nome = "Paulo";

    echo $nome. " tem". $idade. " anos!"
            ou simplesmente,
    echo "$nome tem $idade anos!"

E, simples assim, teremos a saída com as variáveis sendo substituídas por seus valores. Nesse caso, descobriríamos que: Paulo tem 28 anos!

E se, ao invés de aspas duplas "" utilizarmos as aspas simples '', vamos ter como saída apenas texto. Então se utilizássemos as aspas simples no exemplo acima, a saída seria: $nome tem $idade anos!
