Como funciona o PHP

O HTML serve para conteúdo, as CSS são para estilos, o JavaScript é para interatividade. Mas e o PHP?

O PHP tem como objetivo trazer funcionalidades extras para o site, gerando conteúdo de forma dinâmica.

As tecnologias HTML, CSS e JavaScript são consideradas "client side", pois eles funcionam no computador do cliente, através do navegador. O servidor apenas recebe o pedido e entrega o uma cópia do documento solicitado.

O PHP é uma ferramenta "server side", pois ele não deve apenas enviar uma cópia do arquivo PHP para o cliente. Dessa forma, ele tem que ser processado no servidor, por um programa que interpreta os dados do PHP e os transforma em um documento com as tecnologias client side, como o HTML.

Existem várias outras tecnologias Server Side. O PHP é apenas uma delas, e também uma das mais difundidas.

E como vamos aprender o PHP?

Se o PHP é uma linguagem que roda no servidor, como vamos executá-la no nosso computador?

Pra isso, teremos que transformar nossa máquina em duas, trabalhando como cliente e como servidor. Pra isso precisamos de:

  • Apache: Um interpretador de PHP
  • MySql: Um servidor de banco de dados
  • PHP: A estrela do curso, a nossa linguagem de programação

Ao invés de baixar os três programas isoladamente, temos uma opção muito mais versátil. Como as três ferramentas trabalham intimamente ligadas, temos o fornecimento de downloads das três juntas, conhecidos como os AMP (Apache, MySql e PHP). Para usuários Windows, baixamos os programas conhecidos como WAMP, para Linux LAMP e para Mac MAMP, entendeu as siglas?
